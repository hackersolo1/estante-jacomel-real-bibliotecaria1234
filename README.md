# Estante Jacomel - Bibliotecária

Interface usada pela **bibliotecária** do Colégio Jacomel para:
- Gerenciar o catálogo de livros
- Registrar empréstimos e devoluções
- Acompanhar reservas feitas pelos alunos

## Tecnologias
- HTML
- CSS
- JavaScript puro
- Hospedagem estática na Vercel

## Como executar localmente
1. Clone este repositório:
   ```bash
   git clone https://github.com/hackersolo1/estante-jacomel-real-bibliotecaria1234.git
   ```
2. Abra o arquivo `index.html` em qualquer navegador moderno.

## Estrutura básica
- `index.html` – painel principal da bibliotecária
- `biblioteca.css` – estilos da interface administrativa
- `biblioteca.js` – lógica de gerenciamento (livros, alunos, empréstimos, etc.)

## Deploy
O deploy deste front-end é feito na **Vercel**, apontando este repositório como origem.
